## Project Overview
This project analyzes Git as an open source software and demonstrates Linux shell scripting for system analysis.

## Scripts Included

1. system_identity.sh
Displays system information such as user, kernel version, uptime, and distribution.

2. package_inspector.sh
Checks whether Git is installed on the system.

3. disk_auditor.sh
Analyzes directory permissions and disk usage.

4. log_analyzer.sh
Searches for error messages in system log files.

5. manifesto.sh
Generates a short open source manifesto based on user input.

## How to Run Scripts

Make scripts executable:

chmod +x *.sh

Run a script:

./scriptname.sh
